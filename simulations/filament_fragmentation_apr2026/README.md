# Filament Fragmentation Simulations — April 2026

**Author:** Glenn J. White (Open University) / ASTRA System
**Date:** April 2026
**Purpose:** Constrain the plasma β in HGBS filaments via the magnetic tension model

## Key Result

Magnetic tension in filaments with β ≈ 0.76 exactly reproduces the observed
universal λ/W = 2.1 ± 0.1 from the Herschel Gould Belt Survey, across 9 star-forming
regions from Taurus to W3.
