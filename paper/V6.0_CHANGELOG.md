## New Sections (Methods)

### Section 2.9: Cognitive Architecture (~25 lines)
- Central cognitive loop: perceive → reason → discover → learn → reflect
- **Dynamic Knowledge Graph**: NetworkX-based, 17 relation types (causal, theoretical, epistemic), belief propagation, analogy detection, gap identification
- **Neuro-Symbolic Integration**: bidirectional loop — neural pattern discovery → symbolic formalization and vice versa
- **Meta-Cognitive Monitoring**: cognitive state tracking (NORMAL/UNCERTAIN/CONFUSED/OVERCONFIDENT), strategy optimization, error pattern analysis

### Section 2.10: Multi-Agent Scientific Debate (~15 lines)
- Specialized scientific agents (theorist, experimentalist, sceptic, statistician)
- Structured debate: OPENING → REBUTTAL → CLARIFICATION → SYNTHESIS
- Consensus engine: 7 methods (majority, weighted, expertise-weighted, Bayesian, Delphi, Condorcet, Borda)
- Expertise tracking with performance-based task assignment
- Honest caveat: independent contribution not yet isolated via ablation

### Section 2.11: Theory Framework (~15 lines)
- Theory lifecycle: PROPOSED → ACTIVE → TESTING → VALIDATED → PUBLISHED
- Phase 1: Formalization (contradiction detection, dimensional analysis)
- Phase 2: Abstraction (generalization, cross-domain analogy, symmetry scanning)
- Phase 3: Evaluation (abductive reasoning, critical experiment design, self-consistency)
- MCTS for theoretical space exploration (Kocsis & Szepesvári 2006)

### Section 2.12: Autonomous Research Agenda (~10 lines)
- 6 curiosity dimensions: INFORMATION_GAP, NOVELTY_POTENTIAL, FEASIBILITY_BALANCE, SCIENTIFIC_IMPORTANCE, COLLABORATIVE_OPPORTUNITY, RESOURCE_EFFICIENCY
- Research goal lifecycle with human approval gate
- Priority scoring integrating curiosity, importance, feasibility
